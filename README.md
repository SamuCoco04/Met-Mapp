**Weather Station Interface Application**

Met-Mapp is an Android application designed to connect to a Firebase database and visualize real-time meteorological data from weather stations.  
The app allows users to log in, view temperature and humidity readings, and analyze data trends through interactive visualizations.

---

## Features Overview

- Real-time access to weather data from Firebase
- Intuitive and responsive graphical interface
- Statistical data processing (averages, trends, moving averages)
- Interactive visualizations for temperature and humidity
- User login and registration system

---

## Main Required Components

- **Access to Firebase database** to receive real-time weather data
- **Intuitive graphical interface** allowing selection of data to display and date range
- **Data processing module** to calculate averages, moving averages, and trends
- **Visualization of results** through interactive graphs

---

## Layout Plan for Each Application Screen

- **Start Screen:** Presentation of the logo and login
- **Main Screen:** List of available weather parameters (temperature, humidity...) as interactive buttons which, when pressed, open a detailed analysis of the station
- **Station Screen:** Graph of station data (temperature, humidity, date) with average, mode, and other metrics

---

## Data Required for the Application (Data Model)

- Unique identifier for each weather station
- Date of each measurement
- Meteorological variables: temperature, humidity...
- User and display preferences
